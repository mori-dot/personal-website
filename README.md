# 个人官网修改说明

## 如何修改填写位内容

网站中所有需要填写的内容都用【】标注，您可以使用任何文本编辑器（如记事本、VS Code等）打开以下文件进行修改：

### 1. index.html（首页）

- **【你的姓名】**：修改首页标题和网站标题
- **【核心领域】**：修改专注的领域
- **【运营经验】**：填写运营经验相关内容
- **【核心擅长】**：填写核心擅长相关内容
- **【过往成果】**：填写过往成果相关内容
- **【你的微信】**：填写微信联系方式
- **【你的邮箱】**：填写邮箱联系方式

### 2. about.html（关于我页面）

- **【你的姓名】**：修改网站标题
- **【姓名】**：填写个人姓名
- **【运营年限】**：填写运营经验年限
- **【目标站点】**：填写专注的目标市场站点
- **【运营理念】**：填写运营理念
- **【核心能力1-5】**：填写5点核心能力
- **【店铺类型1-2】**：填写店铺类型
- **【负责内容1-2】**：填写负责的内容
- **【达成成果1-2】**：填写达成的成果
- **【你的微信】**：填写微信联系方式
- **【你的邮箱】**：填写邮箱联系方式

## 修改步骤

1. 使用文本编辑器打开需要修改的文件
2. 找到带有【】标注的内容
3. 将【】及其内部的文字替换为您的实际信息
4. 保存文件

## 示例

例如，将【你的姓名】修改为"张三"：

```html
<!-- 修改前 -->
<h1 class="text-4xl md:text-6xl font-bold mb-4">【你的姓名】</h1>

<!-- 修改后 -->
<h1 class="text-4xl md:text-6xl font-bold mb-4">张三</h1>
```

## 注意事项

- 请保持修改后的内容简洁明了
- 不需要修改任何代码结构，只需要替换【】中的内容
- 修改完成后，您可以在本地打开文件预览效果
